# HM-HashChecker

一个专业、高效、现代化的 Windows 文件校验工具。

## 简介

HM-HashChecker 是一款专为 Windows 平台设计的桌面应用程序，支持多种哈希算法，可快速计算和验证文件的完整性。

## 功能特性

- **多算法支持**：支持 12 种主流哈希算法
- **批量处理**：支持同时导入多个文件并发计算
- **拖放操作**：直接拖拽文件到窗口即可导入
- **智能校验**：粘贴官方校验值即可比对文件完整性
- **结果导出**：支持将结果导出为 CSV 格式
- **一键复制**：快速复制单个或全部哈希值
- **实时进度**：显示计算进度和耗时统计
- **文件过滤**：按名称快速筛选文件列表
- **多维排序**：支持按名称、大小、时间、状态排序

## 支持的算法

| 算法 | 输出长度 | 适用场景 | 说明 |
|------|----------|----------|------|
| SHA-256 | 256 位 | 通用校验 | 广泛使用的安全哈希算法 |
| SHA-384 | 384 位 | 高安全性 | 适用于需要更高安全性的场合 |
| SHA-512 | 512 位 | 高性能 | 在 64 位系统上性能优异 |
| SHA3-256 | 256 位 | 高安全性 | SHA-3 标准算法，抗攻击能力强 |
| SHA3-512 | 512 位 | 极高安全 | SHA-3 标准中最强变体 |
| BLAKE2b | 512 位 | 高性能安全 | 比 SHA-3 更快，针对 64 位平台优化 |
| BLAKE2s | 256 位 | 资源受限 | 针对 8-32 位平台优化 |
| SHA-1 | 160 位 | 兼容性 | 较旧算法，仍广泛用于兼容性校验 |
| MD5 | 128 位 | 快速校验 | 经典算法，速度极快 |
| CRC32 | 32 位 | 基本校验 | 循环冗余校验，速度极快 |
| CRC64 | 64 位 | 增强校验 | 比 CRC32 提供更低的碰撞概率 |
| xxHash64 | 64 位 | 极速校验 | 极其快速的非加密哈希算法 |

## 系统要求

- **操作系统**：Windows 10/11 (64位)
- **运行环境**：无需额外安装运行时
- **内存**：建议 4GB 以上
- **磁盘**：约 50MB 可用空间

## 安装和使用

### 安装方式

1. **下载预编译版本**
   - 访问 [Releases 页面](https://github.com/HEIMAO-YS/HM-HashChecker/releases)
   下载最新版本的安装包或便携版

2. **从源码编译**
   ```bash
   git clone https://github.com/HEIMAO-YS/HM-HashChecker.git
   cd HM-HashChecker
   cargo build --release
   ```

### 基本操作

1. **导入文件**
   - 点击「导入文件」按钮选择文件
   - 或直接将文件拖拽到窗口中

2. **选择算法**
   - 在界面顶部选择需要的哈希算法
   - 切换算法后，新导入的文件将使用新算法

3. **开始计算**
   - 导入文件后自动开始计算
   - 可暂停自动计算，手动点击「开始计算」

4. **校验文件**
   - 点击列表中的文件选中
   - 在下方「官方校验比对」区域粘贴官方校验值
   - 点击「比对」按钮查看结果

5. **导出结果**
   - 点击「导出为 CSV」保存计算结果
   - 点击「复制全部」复制所有哈希值

### 快捷操作

| 操作 | 说明 |
|------|------|
| 拖放文件 | 直接将文件拖入窗口导入 |
| 点击文件 | 选中并自动复制哈希值 |
| 空格键 | 切换升序/降序排序 |
| Ctrl+C | 在输入框中快速粘贴校验值 |
